# Проект для поиска нужных монет в онлайн магазинах

Состоит из 2 частей:
1. Backend - API и логика поиска и парсинга предложений о продаже монет из разных источников.
2. Frontend - Расширение для браузера.

## Расширение

Расширение работает в тесной связке с сайтом https://ru.ucoin.net.
Сайт имеет огромную базу монет из множества стран. А также позволяет указывать, какие монеты у тебя уже есть. Таким образом, сайт можно использовать, как систему отслеживания размера и состава своей коллекции.

Расширение работает именно с этим сайтом, дополняя его функционал. Теперь при просмотре информации о конкретной монете, можно будет сразу найти предложения о её продаже в различных онлайн магазинах, увидеть их стоимость и сразу перейти в нужный магазин по ссылке.
